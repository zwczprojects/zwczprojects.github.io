RadioMod for OMSI 2

It's a plugin for OMSI 2 that adds listening to music while driving the bus. It has function of virtual disc changer, supporting 99 of them. Virtual disc can point to: m3u file, directory (with music) or physical disc recorded in Audio CD format. The internet radio gives you access to listening internet stations all over the world. This plugin provides more control over the game built-in radio. It's integrated with the game, including configuration is done via game options.

Current version: 1.0.0
